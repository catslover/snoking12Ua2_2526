# Sno‑King 12U A2 — Hockey Pages Starter (GitHub Pages + Jekyll)

Two‑column header: **logo on white (left ~28%)** + **title/nav on navy with thin green underline (right ~72%)**.  
Edit content via simple Markdown files — GitHub Pages builds it for you.

## Deploy
1. Create a new GitHub repo (public is simplest).
2. Upload all files from this folder.
3. In **Settings → Pages**: Source = *Deploy from a branch*, Branch = `main` (root).

## Edit Content
- Add files to `_tournaments/` and `_events/`. See `about.md` for a full example.

## Customize
- Replace `images/logo.png` with your preferred logo file.
- Colors & layout in `assets/css/site.css` (`--navy`, `--green`, header grid rules).
